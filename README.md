# 基于词图搜索的标题生成算法

## 词图生成

将候选句子输入MSC模型中可以构造出对应话题的词图模型。一般来说，词图是一个加权有向图，图中的节点代表词语，边代表节点与节点之间的连接关系。假设词语A和词语B在输入的句子中有毗邻关系，并且词语A和词语B都在词图中生成各自的节点，那么词图中应该存在一条边从词语A指向词语B。毗邻关系是有向的，由句子中出现在前的词语指向出现在后的词语。词图中，包含了两个重要的节点，即开始节点（STRATNODE）和结束节点（ENDNODE），这两个节点用于标识词图的开始位置和结束位置，不属于之前输入的句子。STARTNODE节点必然指向所有输入句子的句首词语，而ENDNODE节点必然被所有句子的句尾词语指向

## 词图搜素算法

采用改进的集束搜索算法，可调参数为全保留层数和集束宽度

## 实验数据

获取经过话题聚类的一个簇的所有新闻标题，然后生成该簇的标题。
